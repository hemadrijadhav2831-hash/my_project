# Intelligent Healthcare

An AI-powered healthcare system that predicts diseases based on user input and provides simple medical advice.

## Features
- User login and registration
- Disease prediction
- Health advice
- Voice input page
- Result page
- Navbar and clean HTML templates

## Project Structure
intelligent_healthcare/

│     └── partials/

│── static/ (if any)
│── app.py (if included)

## Tech Used

- HTML/CSS

- Machine Learning model
